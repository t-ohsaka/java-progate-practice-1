## 論理演算子

- `&&`：かつ
- `||`：または
- `!`：反転

## if文

```java
if (条件式１) {
    条件式１がtrueの時の処理;
} else if (条件式２) {
    条件式２がtrueの時の処理;
} else {
    上記以外の時の処理;
}
```

## switch文

```java
switch (値) {
    case 値１:
        値の評価==値１ の時の処理
        break;
    case 値２:
    case 値３:
        値の評価==値２ の時の処理
        値の評価==値３ の時の処理
        break;
    default:
        値の評価がどれも一致しなかった場合の処理
        break;
}
```
