
## Boolean型

- `true`
- `false`

## 比較演算子

- `==`
- `!=`
- `<`
- `>`
- `<=`
- `>=`

## 論理演算子

- `&&`：かつ
- `||`：または
- `!`：反転

## if文

```java
if (条件式１) {
    処理１;
} else if (条件式２) {
    処理２;
} else {
    処理３;
}
```

## switch文

```java
switch (条件の値) {
    case 値１:
        処理１;
        break;
    case 値２:
        処理２;
        break;
    default:
        処理３;
}
```
